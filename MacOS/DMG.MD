# Tools download like *.dmg

## Operations

* Download .dmg extension of app
* Click to .dmg
* Move to Applications

## List

* Postman
* Docker desktop
* OpenLens (arm64)
* Commander One
* iTerm
* Tunnelblick (VPN) - optional
* karabiner (modified keyboard [karabiner.json][karabiner_json] ) - optional
